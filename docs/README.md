# Roomie Kit Docs

Roomie Kit is an open-source starter for virtual rooms, livestreaming, video chat, avatars, Supabase, Agora, and mobile apps.

Use these docs to:

- Run the free MIT-licensed starter locally.
- Build full virtual-room products with avatars, rooms, and games.
- Use the media layer by itself for livestreaming or video chat.
- Customize branding, themes, and assets.
- Deploy with Managed Hosting and the `roomie-kit-host` CLI.

## Managed Hosting

Managed Hosting is the paid path for teams that want guided deployment, health checks, starter asset hosting, update tooling, and support for `69 USD/month` with no setup fee.

Start with [Managed Hosting overview](managed-hosting/overview.md), then follow [CLI onboarding](managed-hosting/onboarding.md).
