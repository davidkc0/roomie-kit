# Managed Hosting Overview

Managed Hosting is the paid Roomie Kit deployment layer.

Plan:

- `69 USD/month`
- No setup fee
- Cancel anytime before the next billing period

Managed Hosting provides:

- `roomie-kit-host` CLI access.
- Guided onboarding and deployment.
- Managed starter asset defaults.
- Health checks.
- Update tooling.
- Support.

Customers still own their infrastructure:

- Vercel hosts the web app.
- Supabase hosts auth, database, realtime, and Edge Functions.
- Agora powers media and livestreaming.

Roomie Kit Cloud only handles account state, licenses, entitlements, release metadata, and managed CLI access.
