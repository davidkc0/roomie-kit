# Project Overview

Roomie Kit is a modular starter for realtime social products. It includes:

- A Vite/React web app.
- Supabase auth, database migrations, realtime data, and Edge Functions.
- Agora voice, video, and livestreaming through a provider layer.
- Babylon.js virtual rooms and avatar rendering.
- Capacitor wrappers for iOS and Android.
- Starter assets and local override folders for customer customization.

You can use it as a full virtual-room app, or use the media layer as a livestreaming/video-chat foundation without the 3D spaces.

## Free vs Managed

The open-source starter is MIT licensed and can be self-hosted.

Managed Hosting is optional and adds:

- Guided CLI onboarding.
- Customer-owned Vercel, Supabase, and Agora deployment.
- Managed asset defaults.
- Health checks.
- Update tooling.
- Support.

Managed Hosting is `69 USD/month` with no setup fee.
