# Roomie Web App

This is the Vite React client for the Roomie starter.

```bash
npm install
cp .env.example .env
npm run dev
```

Use `npm run typecheck`, `npm run lint`, and `npm run build` before shipping changes.

Auth callback routes live inside this app:

- `/confirm-email`
- `/reset-password`

Media configuration lives in `src/media` and uses Agora by default.
